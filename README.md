Je suis co-fondateur de **Atarax IT**, une entreprise d’édition de logiciels spécialisée dans le développement de solutions SaaS.

### Produits développés

#### Zendocs

Plateforme de gestion documentaire permettant de gérer la demande, le dépôt et le traitement de justificatifs.

#### MonPiston

Web app d’aide à la recherche d’alternance, stage et emploi, intégrant :

- génération IA de lettres de motivation ;
- suivi des candidatures ;
- relances automatiques ;
- automatisation du processus de recherche.
